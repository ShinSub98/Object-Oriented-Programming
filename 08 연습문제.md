## 객체 활용
**반지름과 이름 필드, 원의 면적을 계산하는 메소드를 가진 Circle 클래스를 작성하여 아래와 같이 출력되도록 Circle 클래스의 객체를 생성하여 프로그램을 작성하라.**

>반지름(10) : 자바피자의 면적은 314.0


==**API**==
```java
public class Circle {  
    String name;  
    int rad;  
  
    public double pi(){  
        return rad*rad*3.14;  
    }  
}
```

==**실행 클래스**==
```java
public class CircleExam {  
    public static void main(String[] args) {  
        Circle circle = new Circle();  
  
        circle.rad = 2;  
        circle.name = "자바도넛";  
  
        System.out.println("반지름("+circle.rad+") : "+ circle.name+"의 면적은 "+circle.pi());  
    }  
}
```



## 생성자 오버로딩
==**API**==
```java
public class Book {  
    String title;  
    String author;  
	  
    public Book(){  
        this(" ", " ");  
    }  
	  
    public Book(String title){  
        this(title, "작자 미상");  
    }  
	  
    public Book(String title, String author){  
        this.title = title;  
        this.author = author;  
    }  
	  
    public void show(){  
        System.out.println("제목: "+title+" 작가: "+author);  
    }  
}
```
매개값을 받지 않아 개발자가 직접 채워줘야 하는 값이 있을 때만 `this(...)
그렇지 않을 때는 `this.xxx = xxx`


==**실행 클래스**==
```java
public class BookExam {  
    public static void main(String[] args) {  
        Book empty = new Book();  
        Book onlyTitle = new Book("춘향전");  
        Book titleAuthor = new Book("어린왕자", "생텍쥐베리");  
		  
        empty.show();  
        onlyTitle.show();  
        titleAuthor.show();  
    }  
}
```



## Setter / Getter
==**API**==
```java
public class Account {  
    private String name;  
    private int balance;  
	  
    public String getName(){  
        return name;  
    }  
	  
    public void setName(String name){  //String으로 받기
        this.name = name;  
    }  
	  
    public int getBalance(){  
        return balance;  
    }  
	  
    public void setBalance(int balance){  //int로 받아서 음수인지 확인하기
        if(balance < 0){  
            this.balance = 0;  
            return;  
        }  
        this.balance = balance;  
    }  
}
```

==**실행 클래스**==
```java
public class AccountExam {  
    public static void main(String[] args) {  
        Account account = new Account();  
		  
        account.setName("Tom");  //set을 통해 저장
        account.setBalance(100000);  
		
		//get을 통해 호출
        System.out.println(account.getName() +" "+ account.getBalance());
    }  
}
```


## super
==**API**==
```java
package practice2;  
  
public class Point {  
    private int x, y;  
	  
    public Point(int x, int y) {  
        this.x = x; this.y = y;  
    }  
	  
    public void showPoint() {  
        System.out.println("(" + x + "," + y + ")");  
    }  
}
```

==**실행 클래스**==
```java
package practice2;  
  
public class ColorPoint extends Point{  
    private String color;  
	  
    public ColorPoint(int x, int y, String color){  
        super(x, y); //부모클래스 생성
        this.color = color;  
    }  
	  
    public void showColorPoint(){  
        System.out.print(color);  
        showPoint();  
    }  
}
```
